# 📰 RSS Feeder

Basit, hızlı ve ücretsiz RSS Feeder.

## 🚀 Özellikler

* Yeni RSS kaynakları (URL) ekleme ve mevcutları yönetme.
* İlgini çeken haberleri daha sonra okumak için "Kaydedilenler" sekmesine alma.
* Belirli zaman aralıklarıyla sayfayı otomatik yenileme.
* Silinen listeleri "Varsayılanları Yükle" butonuyla anında geri getirme.
* Canlı saat ve tarih gösterimi ile sade ve şık bir arayüz.

## 🛠️ Kullanılan Diller

* HTML
* CSS
* JavaScript

## 💻 Nasıl Kullanılır?

1. Ekrandaki **Kaynak Yönetimi** veya **Ekle** butonuna tıkla.
2. Takip etmek istediğin web sitesinin RSS bağlantısını ilgili alana yapıştır.
3. **Tüm Haberler** sekmesine geçerek güncel içerikleri hemen okumaya başla.
